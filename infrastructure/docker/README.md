### Ray node with latest Qiskit

To build image run:
```shell
docker build -f Dockerfile-ray-qiskit -t <IMAGE_NAME> .
```

### Ray node with latest CKT

To build image run:
```shell
docker build -f Dockerfile-ray-ckt -t <IMAGE_NAME> .
```
